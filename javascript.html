Significado de JavaScript

JavaScript é uma linguagem de programação muito utilizada para interação do usuário, manipulação e captura de dados em sites, sendo Front-end e um pouco Back-end (com o Node). Com o DOM, um site inteiro pode ser criado com JavaScript, apesar de ser mais difícil que criar um documento HTML. OBS: Não confundir com liguagem Java. Há semelhança apenas no nome.
